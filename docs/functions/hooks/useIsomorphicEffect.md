# useIsomorphicEffect

Hook conditionally selects either `useLayoutEffect` or `useEffect` based on the environment

```typescript
import { useIsomorphicEffect } from '@sibericancode/reactuse';
```

## Usage
```typescript
useIsomorphicEffect(callback, deps)
```
